# Datalist Reference Multirow

## How to Use

1. Open Apps

2. Choose Form

3. Drag and Drop Form Grid

4. Edit from grid

5. Fill Field

6. Go to Validator

7. Fill load binder field

8. Fill store binder field

9. Go to Load Binder Page

10. Fill Datalist Field

11. Go to Store Binder Page

12. Fill Datalist field

13. Click OK

14. Clik Save

15. Result
