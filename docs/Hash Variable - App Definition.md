### App Definision Hash Variable

---
***Prefix***
```
appDef
```

***Description***

```
To retrieve App Definition informastion
```

***Atrribute***

```
1. #appDef.appId#
2. #appDef.dateCreated#
3. #appDef.dateModified#
4. #appDef.version#
5. #appDef.license#
6. #appDef.name#
7. #appDef.published#

```

***Scope of Use***

- All components within the app.

***Sample Attribute***

```
To display the app definition app.

#appDef.name#

```
